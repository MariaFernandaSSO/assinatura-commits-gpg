# Assinatura de Commits com GPG

Este repositório foi criado como parte de um desafio com o objetivo de configurar a assinatura digital de commits utilizando GPG no Git e validar essa assinatura no GitHub.

O principal objetivo é aumentar a segurança e a confiabilidade do código, garantindo a verificação de identidade do autor por meio da badge verde "Verified" exibida no histórico de commits.

O commit assinado está presente na branch `main`.
